# LAMMPS Data Viewer v0.1.0

Windows 版已打包为 `LammpsDataViewer-windows-x64.zip`。

## 使用方式

1. 下载 `LammpsDataViewer-windows-x64.zip`。
2. 解压整个文件夹。
3. 双击 `LammpsDataViewer.exe` 启动。

打包版已经包含 Python 运行时和必要依赖，使用者无需专门安装或配置 Python。

## 注意

- 请保持 `LammpsDataViewer.exe` 和 `_internal` 文件夹在同一目录下，不要只复制单独的 exe。
- 如需使用 Materials Studio `.car/.mdf/.cor` 转换功能，仍需要本机有 LAMMPS 自带的 `msi2lmp.exe`。
- 首次运行时，Windows 可能会显示安全提醒，选择允许运行即可。
